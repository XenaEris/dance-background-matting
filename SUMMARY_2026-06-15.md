# 6.15 工作总结

## 本次目标

解决 SAM2 合成结果中人物外侧残留黑边的问题。

## 完成内容

1. 确认底部青绿色三角形装饰属于背景，不再作为人物抠像缺陷处理。
2. 定位黑边原因：SAM2 alpha 略厚，把原视频黑色背景边缘带入前景。
3. 新增 alpha 收缩处理：

```text
--alpha-choke
--alpha-choke-feather
```

4. 生成并对比 `choke 1` 和 `choke 2` 两版。
5. 当前推荐使用 `choke 2`。

## 当前成果视频

```text
outputs/6月15日_SAM2去黑边优化成果_10秒.mp4
```

## 推荐参数

```powershell
--refine-mode edge-band `
--guided-radius 5 `
--edge-erode 2 `
--edge-dilate 3 `
--alpha-choke 2 `
--alpha-choke-feather 0.5 `
--foreground-sharpen 0.30 `
--sharpen-radius 1.0
```

## 当前结论

`choke 2` 版本能更好地清理人物外侧黑边，同时没有明显吃掉主体、头冠和手部。后续如果发现细节被过度收缩，可退回 `--alpha-choke 1`。
